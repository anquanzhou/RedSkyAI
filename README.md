### 1.2 设计理念

- **AI 驱动**：利用大语言模型和多智能体协同技术，实现安全测试的智能化
- **企业级**：满足企业级安全需求，支持多租户、权限管理和合规审计
- **全栈覆盖**：从信息收集、漏洞扫描到渗透测试、态势感知的全流程覆盖
- **离线部署**：支持完全离线部署，适配涉密内网环境
- **可视化**：提供直观的安全态势可视化，便于管理层决策

### 1.3 核心价值

- **提高效率**：AI 辅助安全测试，大幅提高测试效率和覆盖率
- **降低门槛**：简化安全测试流程，降低对专业人员的依赖
- **实时感知**：实时监控安全态势，快速响应安全事件
- **合规保障**：满足等保、GDPR 等合规要求
- **成本优化**：减少人工成本，提高安全投入回报率

## 2. 系统架构

### 2.1 整体架构

RedSky AI 采用分层架构设计，确保系统的可扩展性、安全性和可维护性：

1. **前端层**：基于原生轻量 SPA 架构，使用 HTML + Vanilla JavaScript 开发，支持响应式设计
2. **API 层**：提供 RESTful API 和 WebSocket 接口，支持前端与后端的交互
3. **核心服务层**：包含 AI 引擎、安全工具、任务管理等核心功能模块
4. **数据层**：存储用户数据、配置信息、安全测试结果等
5. **外部集成层**：与外部工具、服务和系统的集成接口

### 2.2 技术栈

| 技术类别 | 技术选型 | 说明 |
|---------|---------|------|
| 前端技术 | 原生 HTML + Vanilla JavaScript | 轻量高效，无框架依赖 |
| 构建工具 | Vite | 支持 ES 模块、热更新、按需打包 |
| 样式系统 | Tailwind CSS v3 | 现代化样式重构，自定义深色主题 |
| 可视化引擎 | D3.js v7 | 实现动态拓扑、热力图等安全可视化 |
| 终端交互 | xterm.js | 实现交互式 Web 终端 |
| 后端语言 | Go | 高性能、并发处理能力强 |
| 数据库 | SQLite | 轻量嵌入式数据库，支持离线部署 |
| AI 集成 | 本地模型 + 外部 API | 支持本地部署和云端模型 |

### 2.3 部署架构

#### 单机部署
- 适用于小型环境和测试场景
- 所有组件部署在单台服务器上
- 资源占用低，部署简单

#### 分布式部署
- 适用于大型企业环境
- 组件分布式部署，负载均衡
- 高可用性设计，确保服务稳定

#### 离线部署
- 适用于涉密内网环境
- 所有资源本地打包，无外部依赖
- 支持完全离线运行

## 3. 核心功能模块

### 3.1 智能交互模块

**功能说明**：提供与 AI 智能体的自然语言交互能力，支持用户通过自然语言发起安全测试、查询风险、配置任务。

**核心特性**：
- 多模型支持：支持多种 AI 模型，包括本地模型和云端模型
- 多角色配置：内置渗透测试专家、合规审计员、漏洞分析师、红队攻击手等角色
- 技能插件集成：可调用各种安全工具和技能插件
- 流式响应：提供实时的 AI 响应，增强用户体验
- 上下文管理：智能管理对话上下文，保持对话连贯性

**应用场景**：
- 自然语言发起安全测试
- 智能安全咨询
- 自动化安全配置
- 安全知识问答

### 3.2 安全工具模块

**功能说明**：集成多种安全工具和技能插件，支持安全测试的各个环节。

**核心工具**：
- 技能插件管理：安装、配置、升级安全技能
- WebShell 管理：管理已获取的 WebShell，支持终端交互和文件管理
- 终端交互：提供交互式 Web 终端，支持命令执行
- 文件管理：管理目标系统文件，支持上传下载
- 漏洞管理：发现、分析、跟踪漏洞
- 自动修复：自动生成漏洞修复方案
- 技能守护：确保技能插件的安全性
- 沙箱管理：安全执行可疑代码

**应用场景**：
- 漏洞扫描与修复
- WebShell 管理与操作
- 终端命令执行
- 文件上传下载

### 3.3 信息收集模块

**功能说明**：收集和管理安全相关信息，为安全测试提供数据支持。

**核心功能**：
- FOFA 查询：利用 FOFA 搜索引擎查询资产信息
- 知识库管理：管理安全知识和最佳实践
- 外部 MCP 管理：集成外部安全工具和服务

**应用场景**：
- 资产信息收集
- 安全知识检索
- 外部工具集成

### 3.4 态势感知模块

**功能说明**：提供实时的安全态势可视化，为管理层提供全局安全视图。

**核心特性**：
- 攻防态势大屏：全屏沉浸式态势感知页面
- 资产拓扑：3D 可视化展示网络资产和攻击路径
- ATT&CK 分析：基于 MITRE ATT&CK 框架的攻击链分析
- 实时日志：实时展示系统活动和安全事件
- 可观测性：监控系统性能和安全状态

**应用场景**：
- 安全态势监控
- 攻击路径分析
- 安全事件响应
- 管理层决策支持

### 3.5 数据管理模块

**功能说明**：管理系统数据，包括智能体记忆、知识库、历史记录等。

**核心功能**：
- 记忆管理：管理智能体的短期和长期记忆
- 知识库管理：管理安全知识库和规则
- 历史记录：存储和查询历史测试记录
- 文件上传：支持文件上传和管理
- 导入导出：支持数据的导入和导出

**应用场景**：
- 智能体行为优化
- 知识积累和复用
- 历史数据查询
- 数据备份和迁移

### 3.6 系统管理模块

**功能说明**：管理系统配置、用户权限、租户设置等。

**核心功能**：
- 租户管理：创建和管理租户，实现多租户隔离
- 用户权限：配置用户角色和权限，实现细粒度权限控制
- 模型配置：配置 AI 模型参数和路由
- 配额管理：管理系统资源使用配额
- 认证管理：配置认证方式，支持 SSO 集成
- 集成管理：配置与外部系统的集成
- 合规配置：配置合规规则和审计设置

**应用场景**：
- 多租户管理
- 权限控制
- 系统配置
- 合规审计

### 3.7 企业扩展模块

**功能说明**：提供企业级的高级功能，满足企业复杂的安全需求。

**核心功能**：
- 红队测试：模拟真实攻击场景，进行全面的安全测试
- 报告引擎：自动生成专业的安全测试报告
- 幻觉检测：检测和标记 AI 生成内容中的幻觉
- HITL 审批：对高危操作进行人工审批
- 护栏管理：配置 AI 安全护栏，防止有害操作
- Hook 管理：配置系统生命周期钩子，实现自定义扩展

**应用场景**：
- 企业安全评估
- 合规报告生成
- 安全操作审批
- 系统自定义扩展
